# Omnifit

Plataforma de analytics esportivo multiesportiva. Consolida treinos de Corrida,
Musculação, Ciclismo, Natação e HYROX numa Home única, com inteligência sobre a carga
esportiva total do atleta — não apenas dashboards isolados por modalidade.

Hoje o módulo **Corrida** é o único totalmente implementado (registro de treinos,
Dashboard, Analytics com evolução de pace, pace x FC, consistência, recordes pessoais e
comparação de ciclos). Os demais módulos aparecem na navegação como "em breve".

## Stack

React + Vite + Tailwind CSS 4 + Recharts + lucide-react.

## Desenvolvimento

```
npm install
npm run dev
npm run lint
```
